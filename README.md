#StockExchange 
take-home assignment

##Overview

Simple StockExchange implemented with Akka for Java.

### packages
-   api: public interfaces
-   stock: stock model
-   util: money and parsing
-   gbce: GBCE listing
-   simpleimpl: implementation based on Akka for Java

### prerequisities
-   JDK 1.8+
-   Scala 2.11.x
-   SBT 0.13.8

### build
```
$sbt test:compile
```

### test
```
$sbt test
```

### run
```
$sbt run
```
